Vue d'ensemble

Le module SEO Clients vous permet de centraliser et piloter le référencement naturel de tous vos clients depuis une seule interface. Il est conçu pour les agences et consultants qui gèrent plusieurs sites en parallèle.

Accéder au module SEO

Le module SEO est accessible depuis le module Prospection de S@FE CRM.

1

Ouvrir la Prospection

Depuis le menu principal de S@FE CRM, cliquez sur Prospection terrain.

2

Accéder au module SEO

Dans la barre de navigation de la prospection, cliquez sur le bouton 🔍 SEO Clients. Vous serez redirigé vers modules/SEO/module-seo-clients.html.

3

Sélectionner un client

Dans la colonne de gauche, utilisez la recherche ou faites défiler la liste pour trouver votre client. Cliquez sur son nom pour charger son espace SEO.

Suivi des mots-clés

La vue 🔑 Mots-clés permet de suivre manuellement les positions Google de chaque mot-clé cible d'un client, avec historique d'évolution et volume de recherche mensuel.

Récapitulatif du domaine

  • Gains — mots-clés dont la position s'est améliorée (↑)

Les 5 statuts disponibles pour chaque point :

  • ⚪ Non vérifié — point pas encore évalué (statut par défaut)
  • ✅ Conforme — le point est correctement mis en œuvre
  • 🟡 Partiel — partiellement conforme, compte pour 0,5 point
  • ❌ Non conforme — problème identifié, action requise
  • ➖ N/A — non applicable à ce client ou ce site

Chaque point dispose d'un champ "Note" pour ajouter des observations détaillées (observations, URLs concernées, recommandations spécifiques).

Sauvegarder l'audit

Cliquez sur "💾 Sauvegarder" pour enregistrer l'état de tous les points en base de données. Le score global se met à jour automatiquement à chaque sauvegarde.

Exporter l'audit en PDF

Cliquez sur "📄 PDF" pour générer un document PDF de l'audit. Le fichier est téléchargé sous le nom audit-seo-[NomClient]-[Domaine].pdf et contient :

  • Le score global et les scores par catégorie
  • L'état de chacun des 25 points
  • Les notes associées à chaque point
  • Les recommandations générées depuis les points non conformes
⚠️

Pensez à sauvegarder avant d'exporter en PDF pour que le document reflète bien la dernière version de l'audit.

Système de score SEO

Le score SEO est calculé automatiquement à partir des résultats de l'audit. Il reflète la santé globale du référencement du client selon une pondération par catégorie.

Formule de calcul

Pour chaque catégorie, le score est calculé comme suit :

  • Point ✅ Conforme = 1 point
  • Point 🟡 Partiel = 0,5 point
  • Point ❌ Non conforme = 0 point
  • Point ➖ N/A = exclu du calcul
  • Point ⚪ Non vérifié = 0 point

Le score de catégorie = (somme des points) ÷ (nombre de points évalués) × 100.

Pondération par catégorie

Le score global est une moyenne pondérée des 5 catégories :

⚙️ Technique
30%
📝 On-Page
25%
✍️ Contenu
20%
📍 SEO Local
15%
🔗 Backlinks
10%

Code couleur des scores

  • 🟢 Bon — ≥ 80% — le site est globalement bien optimisé
  • 🟡 À améliorer — 50 à 79% — des actions ciblées peuvent faire progresser le score
  • 🔴 Faible — < 50% — des lacunes importantes nécessitent une action urgente

Ce score est affiché :

  • Dans la liste de contacts (colonne latérale) avec une barre de progression
  • Dans le sous-menu client sous le nom du contact
  • Sur le domaine sélectionné (pilule dans le subnav)
  • Dans le tableau de bord global (score moyen et classement)
  • En grand format dans la vue Audit SEO

Connecteurs et intégrations

Le module SEO peut se connecter à des outils externes pour enrichir les données. Les connecteurs sont gérés depuis le tableau de bord et s'appliquent à l'ensemble des clients.

Connecteurs disponibles

🔍
Google Search Console

Importe automatiquement les positions Google depuis GSC, sans saisie manuelle. Nécessite une autorisation OAuth.

📍
Google My Business

Récupère les avis clients et les statistiques de la fiche GMB. Simplifie la section "SEO Local" de l'audit.

📈
SE Ranking

Accède aux rapports d'audit technique et au profil de backlinks depuis la plateforme SE Ranking.

🤖
Connecteurs IA (Groq / Grok / Claude / Mistral)

Alimente l'assistant IA avec un vrai modèle de langage. Sans connecteur, l'assistant fonctionne en mode simulation.

💡

Les connecteurs sont configurables depuis le tableau de bord principal de S@FE CRM, dans la section Paramètres → Connecteurs. La clé API de chaque service est stockée de façon sécurisée dans Supabase.
